# Html 学习笔记

## CSS - padding

css 中，padding 用于设置元素在四个方向上的边距

````
padding:10px 5px 15px 20px;     上、右、下、左边距
padding:10px 5px 15px;          上、左右、下边距
padding:10px 5px;               上下、左右边距
padding:10px;                   上下左右边距
````
